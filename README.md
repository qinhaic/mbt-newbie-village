# MBT新手村

心智化治疗（MBT）系统学习网站。

基于 Bateman & Fonagy 的 MBT 框架，从核心概念到临床实践逐级深入。

## 六阶学习路径

| 关卡 | 内容 |
|------|------|
| L1 | 什么是心智化 — 定义、Fonagy的理论、四个维度 |
| L2 | 三个非心智化模式 — 精神等同、假装模式、目的论模式 |
| L3 | 三种心智位 — I-mode、Me-mode、We-mode |
| L4 | 治疗师姿态 — 核心态度、17项干预原则、8大能力领域 |
| L5 | MBT-ACS 自检 — 依从性与能力量表简化版 |
| L6 | 案例整合实践 — 逐段标注的治疗对话 |

## 技术栈

- 纯静态 HTML/CSS/JS
- Tailwind CSS CDN
- 部署于 GitHub Pages

## 内容来源

- Bateman & Fonagy. *人格障碍的心智化治疗*
- Karterud & Bateman (2012). *MBT Adherence and Competence Scale*
- Bateman, Bales & Hutsebaut (2023). *MBT Quality Manual*
- 秦海老师的 MBT 资料库
